# Описание структуры порфтолио

1. 'Data Analyse'. 

Решение задачи предсказания стоимости подержанных автомобилей. 
Внутри файл в формате .ipynb (Jupiter Notebook) с полным циклом анализа данных, а точнее его следующие этапы:
	1. Загрузка данных
	2. Data Preparation
	3. Feature engineering
	4. Modelling
	5. Results


2. 'Airflow'

В данной папке содержится разработанный и протестированный пайплайн обучения ML-модели по предсказанию стоимости подержанных автомобилей.


3. 'Power BI'

Здесь лежит файл с визуализацией набора статистических показателей о различных странах.
