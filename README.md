# Data / BI Analytics Portfolio  

В данном репозитории представлены учебные проекты, моделирующие реальные бизнес-сценарии и демонстрирующие примеры циклов работы с данными: от анализа до автоматизации, а также визуализации.

---

## Структура репозитория

- Data Analysis — аналитическое исследование данных
- Airflow — ETL-процесс для автоматизации обработки данных
- Power BI — интерактивный BI-дашборд

---

# Проект 1–2: Анализ рынка подержанных автомобилей и ETL-процесс

Для запуска онлайн-платформы по продаже подержанных автомобилей необходимо:

- определить факторы, влияющие на стоимость автомобиля;  
- выявить закономерности в данных;
- автоматизировать процесс подготовки данных для аналитики.

Проект состоит из двух частей: аналитического исследования и построения ETL-процесса.

---

## Часть 1. Анализ данных (Data Analysis)

Проведение факторного анализа формирования цены автомобиля и построение модели прогнозирования его стоимости.

### Этапы работы

1. Очистка данных (обработка пропусков и выбросов); 
2. Предобработка и кодирование признаков; 
3. Exploratory Data Analysis (EDA);
4. Анализ корреляций;
5. Построение модели прогнозирования;
6. Интерпретация результатов и выбор модели с наилучшим результатом предсказания.

---

## Часть 2. ETL-процесс (Airflow)

Автоматизация процесса загрузки, очистки и подготовки данных. 
Обучение базовых ML-моделей, а также выбор модели с наилучшим результатом предсказания цены.

---

# Проект 3: BI-дашборд в Power BI

Разработка интерактивного дашборда для анализа ключевых статистических показателей о различных странах.
